Comparison

The purpose was to analyze weather data to determine how latitude may affect a cities' weather. In order to accomplish this, the data collected for this project was cities' latitude, longitude, temperature, windspeed, cloud coverage, and humidity. The data collect from the api was then converted to a csv file inorder to be cleaned and manipulated in python pandas library. The Python Matplotlib library was used to create visualizations for the analysis of the data.
